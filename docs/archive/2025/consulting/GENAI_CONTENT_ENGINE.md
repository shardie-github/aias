### Step 2: Systems Thinking Analysis

**6-Perspective Analysis:**

#### 1. SEO Perspective
**Analyzes:**
- Keywords and keyword density
- Meta tags and descriptions
- Heading structure (H1, H2, H3)
- Content length and optimization
- Search intent alignment
- Competitive analysis

**Identifies:**
- Missing keywords
- SEO gaps
- Optimization opportunities
- Content structure issues

---

#### 2. User Experience Perspective
**Analyzes:**
- Readability scores
- Content flow and navigation
- User journey mapping
- Accessibility compliance
- Mobile optimization
- Conversion paths

**Identifies:**
- UX friction points
- Readability improvements
- Navigation gaps
- Conversion barriers

---

#### 3. Content Structure Perspective
**Analyzes:**
- Information architecture
- Content hierarchy
- Logical flow
- Content organization
- Topic clustering
- Content relationships

**Identifies:**
- Structural improvements
- Organization gaps
- Flow issues
- Relationship opportunities

---

#### 4. Conversion Perspective
**Analyzes:**
- Call-to-action placement
- Conversion opportunities
- Value proposition clarity
- Trust signals
- Social proof
- Urgency elements

**Identifies:**
- Missing CTAs
- Conversion barriers
- Value communication gaps
- Trust signal opportunities

---

#### 5. Technical SEO Perspective
**Analyzes:**
- Page speed optimization
- Mobile responsiveness
- Structured data (Schema.org)
- Internal linking
- Image optimization
- Technical requirements

**Identifies:**
- Technical issues
- Performance gaps
- Schema markup opportunities
- Internal linking improvements

---

#### 6. Systems Architecture Perspective
**Analyzes:**
- How content fits website system
- Content relationships
- Site-wide consistency
- User journey integration
- Content ecosystem
- Cross-page optimization

**Identifies:**
- System integration issues
- Consistency gaps
- Journey breaks
- Ecosystem opportunities

   - Meta tags

   - CTAs

   - Social proof

### Step 4: Automated Website Creation

**Engine automatically:**
- Creates/updates website pages

- Integrates with existing content
