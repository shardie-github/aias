# Production Framework Operations

## Quick Start

```bash
npm run ops init
npm run ops doctor
npm run ops check
```

## Commands

- `ops doctor` - Comprehensive health checks
- `ops init` - Initialize framework
- `ops check` - Quick validation
- `ops release` - Execute release
- `ops snapshot` - Create DB snapshot
- `ops restore` - Restore from snapshot
- `ops rotate-secrets` - Rotate secrets
- `ops sb-guard` - RLS audit
- `ops test:e2e` - E2E tests
- `ops benchmark` - Performance benchmarks
- `ops lintfix` - Auto-fix linting
- `ops docs` - Generate docs
- `ops changelog` - Generate changelog
